# 💰🛢️📈 Cross-Market Analysis: Crypto, Oil & Stocks

This project compares Cryptocurrency, Oil, and Stock markets using
SQL analytics and a Streamlit dashboard.

## Tech Stack
- Python
- SQL (SQLite)
- Streamlit
- Pandas
- CoinGecko API
- Yahoo Finance

## How to Run

pip install -r requirements.txt
